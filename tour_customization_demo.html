How It Works

Step 1: User selects days and locations in cust_new.php
Step 2: Form submits to booking_summary.php with itinerary data
Step 3: SmartTravelPlanner calculates distances and analyzes routes
Step 4: Page displays route analysis, alternatives, and recommendations
Step 5: User sees optimized itinerary before confirming booking

SmartTravelPlanner Methods:

getDistance('Galle', 'Jaffna') // Returns: ~800 km getTravelHours('Galle', 'Jaffna') // Returns: ~18 hours isHopFeasible('Galle', 'Jaffna') // Returns: false (> 6 hrs) getNearbyLocations('Galle', 80) // Returns: [Matara, Mirissa, ...]
