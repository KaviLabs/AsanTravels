# AsanTravels UI/UX Strategy and Wireframe

## 1. Product Focus
AsanTravels is a Sri Lanka travel brand focused on personalized tours, memorable destinations, and easy booking. The experience should feel calm, confident, and trustworthy within seconds of landing on the site.

## 2. Core UX Goals
- Communicate value immediately in the first 3 seconds.
- Keep the journey simple for first-time visitors.
- Make every page feel consistent and premium.
- Support better SEO through semantic structure and performance-conscious layout choices.

## 3. 10-Second Engagement Rule
### Above-the-fold experience
The hero section should include:
- A clear headline: “Personalized Sri Lanka Tours with Local Experts”
- One short supporting line explaining the benefit
- One primary CTA: “Plan Your Trip”
- One secondary CTA: “Explore Packages”
- A small trust row with proof points such as “10+ years experience”, “700+ tours”, and “24/7 support”

### Interaction clarity
Users should immediately understand:
- Where to click next
- What the brand offers
- How to browse destinations or request a custom itinerary

## 4. Design System
### Color palette
Use the existing brand palette as the foundation:
- Primary Blue: #13357B
- Deep Navy: #0F2A62
- Light Background: #F2F2F2
- Dark Text: #14141F
- Accent Gold: #dfc384

### Typography
- Headings: Jost, bold and confident
- Body text: Roboto, clean and readable
- Type scale:
  - H1: 48px desktop / 32px mobile
  - H2: 32px desktop / 24px mobile
  - H3: 24px
  - Body: 16px
  - Small text: 14px

### Spacing system
Use an 8px grid:
- 8px, 16px, 24px, 32px, 48px, 64px

### Reusable components
- Primary button
- Secondary button
- Navigation bar
- Card components for packages and destinations
- Input fields for booking and newsletter forms
- Icon and trust badge blocks

## 5. Layout Architecture
### Header
- Logo on the left
- Navigation links: Home, About, Packages, Contact
- CTA button: “Plan Your Trip”

### Hero Section
- Large image or optimized background with a soft overlay
- Headline, subheadline, CTA buttons
- Trust badges beneath the CTA

### Main Content
- Popular destinations grid
- Featured packages/cards
- Why choose AsanTravels section
- Simple booking or inquiry section
- Testimonials/social proof

### Footer
- Brand summary
- Quick links
- Contact details
- Social links

## 6. Semantic HTML Structure
The page should be built with:
- header
- nav
- main
- section
- article
- form
- footer

This supports accessibility and improves SEO relevance.

## 7. Mobile-First Strategy
- Prioritize one clear CTA on mobile screens.
- Keep touch targets at least 44px high.
- Use generous spacing and readable text sizes.
- Avoid heavy top-of-page media that slows loading.
- Ensure contrast meets accessible standards for text and buttons.

## 8. SEO and Performance Guidance
- Use optimized hero imagery and lazy-load non-essential media.
- Keep LCP-friendly content above the fold.
- Avoid layout-shifting elements near the top of the page.
- Use descriptive headings and meaningful alt text.
- Keep the content hierarchy clear for crawlers and users.

## 9. Wireframe Structure
### Desktop Wireframe
1. Header
2. Hero with headline, CTA, trust row
3. Featured destinations section
4. Popular packages section
5. Why choose us section
6. Testimonials section
7. Booking/inquiry CTA
8. Footer

### Mobile Wireframe
1. Compact header with logo and menu
2. Hero stacked vertically
3. One CTA button first
4. Destination cards in a single column
5. Trust and testimonials stacked for readability
6. Sticky or simple footer navigation

## 10. Suggested Page Flow
Home -> Packages -> Custom Booking -> Confirmation

This flow keeps the user journey short and conversion-oriented.
